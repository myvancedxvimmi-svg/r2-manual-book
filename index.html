📌 Manual Book — Pusat Pelaporan Kerusakan Resmi R2PCKGZ1

Bot pencatat: 24/7  •  Teknisi on-call: Senin–Jumat 08.00–17.00 WIB

1. SOP Pelaporan Kerusakan

  1. Pastikan posisi mesin / line sudah benar sebelum melapor.
  2. Kirim laporan langsung ke bot dengan format standar (lihat Aturan Format).
  3. Tunggu balasan ✅ Dicatat dari bot sebagai bukti terekam.
  4. Cek rekapan via /review; kirim ke rekapan resmi via /kirim.
  5. Tutup shift dengan /selesai.

2. Aturan Format

Satu kerusakan = satu baris:

Line [N] Mc [No] : [Masalah]

Contoh:

Line 10 Mc 1 : Mati lampu

Multi-line:

Line 1-5 Mc 2 : Error sensor suhu

3. Perintah Cepat

  • /review — lihat rekapan laporan.
  • /kirim — kirim rekapan ke grup rekapan resmi.
  • /selesai — arsipkan & tutup shift.
  • /help — tampilkan pesan bantuan.

4. Ketentuan Grup

  • Topik hanya laporan kerusakan & diskusi mesin R2PCKGZ1.
  • Dilarang spam, off-topic, dan konten tidak pantas.
  • Dilarang kirim file / link mencurigakan.
  • Hormati operator & teknisi.
  • Pelanggaran berulang = peringatan → kick.
